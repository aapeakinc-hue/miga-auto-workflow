## 🎯 客户分类管理策略

### 客户分层标准

#### A类客户（VIP客户）
**定义**:
- 已成交，单笔订单>$10,000
- 有长期合作意向
- 回复及时，沟通顺畅

**管理策略**:
- 专属客户经理
- 优先发货
- 专属折扣
- 定期回访（每月1次）
- 赠送样品

**跟进频率**: 每周1次
**转化目标**: 100%续单

---

#### B类客户（重点客户）
**定义**:
- 已回复，有意向
- 单笔订单潜力$5,000-$10,000
- 正在评估中

**管理策略**:
- 主动跟进
- 提供产品目录
- 寄送样品
- 有限折扣（10%）
- 视频会议沟通

**跟进频率**: 每3天1次
**转化目标**: 50%转化为A类

---

#### C类客户（潜在客户）
**定义**:
- 已发送开发邮件
- 未回复
- 有潜在需求

**管理策略**:
- 定期发送产品更新
- 节日问候
- 提供案例分享
- 自动化邮件序列

**跟进频率**: 每2周1次
**转化目标**: 30%转化为B类

---

#### D类客户（无效客户）
**定义**:
- 邮箱无效
- 明确拒绝
- 无需求

**管理策略**:
- 标记为无效
- 定期清理（6个月）
- 保存联系方式（备用）

## 📊 CRM系统设计

### 数据库结构

#### 客户基本信息表
```sql
CREATE TABLE customers (
  id SERIAL PRIMARY KEY,
  company_name VARCHAR(255) NOT NULL,
  contact_name VARCHAR(255),
  email VARCHAR(255) UNIQUE NOT NULL,
  phone VARCHAR(50),
  website VARCHAR(255),
  country VARCHAR(100),
  industry VARCHAR(100),
  customer_type VARCHAR(50),  -- A, B, C, D
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

#### 互动记录表
```sql
CREATE TABLE interactions (
  id SERIAL PRIMARY KEY,
  customer_id INTEGER REFERENCES customers(id),
  interaction_type VARCHAR(50),  -- email, call, meeting
  interaction_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  notes TEXT,
  followup_date DATE,
  status VARCHAR(50),  -- pending, completed
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

#### 订单记录表
```sql
CREATE TABLE orders (
  id SERIAL PRIMARY KEY,
  customer_id INTEGER REFERENCES customers(id),
  order_date DATE NOT NULL,
  total_amount DECIMAL(10,2),
  status VARCHAR(50),  -- pending, shipped, completed
  notes TEXT,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

#### 产品兴趣表
```sql
CREATE TABLE product_interests (
  id SERIAL PRIMARY KEY,
  customer_id INTEGER REFERENCES customers(id),
  product_category VARCHAR(100),  -- candelabra, crafts
  interest_level VARCHAR(50),  -- high, medium, low
  notes TEXT,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 🔄 客户生命周期管理

### 客户生命周期阶段

#### 阶段1: 潜在客户（Lead）
**时长**: 0-30天
**行动**: 发送开发邮件
**转化率目标**: 20%进入下一阶段

#### 阶段2: 意向客户（Prospect）
**时长**: 30-60天
**行动**: 跟进沟通，提供资料
**转化率目标**: 50%进入下一阶段

#### 阶段3: 谈判客户（Opportunity）
**时长**: 60-90天
**行动**: 价格谈判，样品确认
**转化率目标**: 30%进入下一阶段

#### 阶段4: 成交客户（Customer）
**时长**: 90天+
**行动**: 订单执行，售后服务
**续单率目标**: 60%

#### 阶段5: 忠诚客户（Loyal Customer）
**时长**: 12个月+
**行动**: 长期合作，推荐新客户
**推荐率目标**: 20%
