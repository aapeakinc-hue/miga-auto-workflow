## 📊 目标设定示例

### 市场数据驱动

```
美国市场规模: $50,000,000
目标市场份额: 5%
→ 年度收入目标: $2,500,000
→ 年度成交客户: 125个（平均订单$20,000）
→ 年度意向客户: 625个（转化率20%）
→ 年度潜在客户: 6,250个（意向率10%）
```

### 月度目标分解（考虑季节性）

```
年度目标: 6,250个潜在客户
1月: 500个（权重8%）
...
10月: 625个（权重10% - 旺季）
11月: 688个（权重11% - 旺季高峰）
12月: 563个（权重9% - 旺季延续）
```

### 目标调整机制

```
月度达成率: 130%
→ 下月目标提高15%

月度达成率: 75%
→ 下月目标降低10%

月度达成率: 50%
→ 下月目标降低20%
```
