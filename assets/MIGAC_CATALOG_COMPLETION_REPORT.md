## 🏷️ 新型号系统特点

### 优点

1. **清晰明了**
   - 统一的命名格式，易于识别和管理
   - 每个字符都有明确的含义，不需要记忆

2. **便于扩展**
   - 产品类型编号支持 001-999，可容纳大量产品
   - 新增系列和颜色只需添加新的代码

3. **国际通用**
   - 采用英文代码，便于国际客户理解
   - 符合行业惯例，提升专业形象

4. **信息丰富**
   - 从型号可以直接读取产品的关键信息
   - 便于快速筛选和查找产品

5. **便于管理**
   - 适合 ERP 系统和数据库管理
   - 便于生成条形码和二维码

### 型号示例

```

│ │   │   │   └─ Natural（无色透明）
│ │   │   └───── 83cm 高度
│ │   └───────── 第 001 号产品
│ └───────────── CA = Crystal Candelabra
└─────────────── MG = MIGAC 品牌
```
