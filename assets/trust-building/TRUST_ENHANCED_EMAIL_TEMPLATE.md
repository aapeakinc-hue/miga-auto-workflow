## 🎯 邮件主题优化

### A/B测试主题
```
版本A（标准版）：
"Crystal Candle Holders for {客户公司名称} | Factory-Direct Pricing"

版本B（价值导向）：
"30-40% Below Market: Premium Crystal Candle Holders for {客户公司名称}"

版本C（信任导向）：
"Trusted by 200+ Hotels & Wholesalers | Crystal Crafts for {客户公司名称}"

版本D（紧急导向）：
"Limited Time Offer: Free Crystal Samples for {客户公司名称}"

版本E（个性化）：
"Saw Your {提到客户产品} | Perfect Crystal Match from Migac"
```

### 主题最佳实践
```
✓ 包含客户公司名称（个性化）
✓ 包含核心产品（水晶烛台）
✓ 包含价值主张（价格、质量、信任）
✓ 长度控制在6-8个单词
✓ 避免过度营销语言
✓ 避免使用大写字母（除了第一个词）
✓ 避免使用特殊符号（除了 | 和 -）
```

## 📧 邮件发送最佳实践

### 发送时间
```
最佳发送时间（客户当地时间）：
- 周二至周四（避开周一和周五）
- 上午 9:00 - 11:00
- 下午 2:00 - 4:00

避开时间：
- 周末
- 节假日
- 客户国家的公共假期
- 客户公司的假期
```

### 发送频率
```
✅ 第一封：初始介绍邮件
✅ 3天后：跟进邮件（提醒）
✅ 7天后：价值邮件（产品亮点）
✅ 14天后：优惠邮件（限时优惠）
✅ 30天后：最后邮件（是否继续）

停止条件：
- 客户回复（停止发送，转入跟进流程）
- 客户拒绝（停止发送，标记为不感兴趣）
- 5封邮件后无回复（停止发送，标记为无响应）
```

### 邮件格式
```
✓ 使用纯文本格式（兼容性最好）
✓ 使用HTML格式（视觉效果好，兼容性稍差）
✓ 使用响应式设计（适配手机）
✓ 邮件宽度：600-800像素
✓ 字体：Arial, Helvetica, sans-serif
✓ 字号：14-16px
✓ 行间距：1.5倍
```

## 📊 邮件效果追踪

### 追踪指标
```
打开率（Open Rate）：
- 目标：35-45%
- 行业平均：20-30%
- 优化空间：个性化主题 + 信任元素

点击率（Click-Through Rate）：
- 目标：3-5%
- 行业平均：1-3%
- 优化空间：清晰CTA + 价值主张

回复率（Reply Rate）：
- 目标：8-12%
- 行业平均：1-3%
- 优化空间：个性化 + 信任元素

转化率（Conversion Rate）：
- 目标：2-3%
- 行业平均：0.5-1%
- 优化空间：跟进流程 + 信任建设
```

### A/B测试建议
```
测试变量：
1. 邮件主题（版本A vs 版本B）
2. 开场白（个性化 vs 通用）
3. 信任元素（多 vs 少）
4. CTA位置（顶部 vs 底部）
5. 发送时间（上午 vs 下午）

测试方法：
- 同时发送两个版本给相同数量的客户
- 每个版本至少100封邮件
- 统计7天内的数据
- 选择效果最好的版本
```

## 📝 使用指南

### 在工作流中使用

#### 1. 读取客户网站内容
```python
# 在 customer_search_node.py 中
# 提取客户网站的具体内容
# 例如：产品线、特色、公司介绍等

website_content = extract_website_content(customer_website)
customer_specialty = identify_customer_specialty(website_content)
```

#### 2. 生成个性化开场白
```python
# 在 email_generate_node.py 中
# 根据客户特色生成个性化开场白

if customer_specialty:
    opening = f"I was particularly impressed by your {customer_specialty}."
else:
    opening = "I was impressed by your selection."
```

#### 3. 根据客户类型选择模板
```python
# 在 email_generate_node.py 中
# 根据客户类型选择对应的邮件模板

if customer_type == "hotel":
    template = HOTEL_EMAIL_TEMPLATE
elif customer_type == "wholesaler":
    template = WHOLESALER_EMAIL_TEMPLATE
elif customer_type == "event_planner":
    template = EVENT_PLANNER_EMAIL_TEMPLATE
else:
    template = STANDARD_EMAIL_TEMPLATE
```

#### 4. 添加信任元素
```python
# 在 email_generate_node.py 中
# 自动添加信任元素

trust_elements = {
    "company_background": COMPANY_PROFILE_SUMMARY,

    "quality_guarantee": QUALITY_ASSURANCE_SUMMARY

    company_name=customer_company_name,
```
