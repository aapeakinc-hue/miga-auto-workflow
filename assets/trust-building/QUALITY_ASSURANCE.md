## 🔬 质量检验流程

### 全流程质检体系

#### 1. 原材料检验
```
检验项目：
✓ 水晶原材料的透明度
✓ 无气泡、无杂质
✓ 铅含量检测（<0.06%）
✓ 硬度测试
✓ 折射率测试

合格标准：
- 透明度 > 99%
- 无可见杂质
- 铅含量 < 0.06%
- 硬度 > 7（莫氏硬度）
- 折射率 1.5-1.6
```

#### 2. 生产过程检验
```
检验环节：
✓ 切割尺寸精度检查
✓ 打磨平整度检查
✓ 抛光光泽度检查
✓ 雕刻精度检查
✓ 组装牢固度检查

检验标准：
- 尺寸误差 ±0.5mm
- 表面粗糙度 < 0.1μm
- 光泽度 > 95%
- 雕刻深度偏差 < 0.1mm
- 组装牢固无松动
```

#### 3. 成品检验
```
检验项目：
✓ 外观质量（无划痕、无裂纹）
✓ 尺寸规格（符合设计要求）
✓ 功能测试（烛台稳定性等）
✓ 包装检查（无破损）
✓ 标识检查（标签、Logo清晰）

合格标准：
- 外观完美无瑕疵
- 尺寸误差 < 1%
- 功能正常稳定
- 包装完好无损
- 标识清晰准确
```

#### 4. 出货前终检
```
检验内容：
✓ 数量核对
✓ 质量复核
✓ 包装完整性检查
✓ 订单匹配检查
✓ 物流准备检查

执行标准：
- 100%全检，不抽检
- 不合格品坚决不出货
- 每批产品附质检报告
- 记录每批产品的检验数据
```

### 正式订单售后服务

#### 1. 质量问题处理
```
质量问题定义：
✓ 产品有明显瑕疵（划痕、裂纹、破损）
✓ 产品与订单规格不符
✓ 产品功能缺陷
✓ 包装破损导致产品损坏

处理方案：
① 轻微质量问题（不影响使用）
   - 提供部分退款（5-15%）
   - 客户保留产品

② 严重质量问题（影响使用）
   - 免费更换新产品
   - 承担来回运费
   - 5-10天内完成更换

③ 无法修复的质量问题
   - 全额退款
   - 承担来回运费
   - 道歉并赠送下次订单折扣
```

#### 2. 退换货政策
```
退货条件：
✓ 收货后7天内提出退货
✓ 产品未使用、未损坏
✓ 包装完好、配件齐全
✓ 提供退货原因说明

退货流程：
1. 联系客服申请退货
2. 提供退货原因和照片
3. 客服审核（1-2个工作日）
4. 寄回产品（运费自理）
5. 收到产品后5天内退款

退货费用：
✓ 退款金额：全额（不含运费）
✓ 退款时间：5个工作日
✓ 退款方式：原路返回

换货条件：
✓ 收货后10天内提出换货
✓ 产品未使用、未损坏
✓ 包装完好、配件齐全
✓ 换货产品与原订单等值

换货流程：
1. 联系客服申请换货
2. 提供换货原因和照片
3. 选择换货产品
4. 寄回原产品（运费自理）
5. 新产品7-10天内发货

换货费用：
✓ 差价处理：多退少补
✓ 运费：客户承担
```
