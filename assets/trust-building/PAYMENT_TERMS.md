## 📝 付款流程

### T/T付款流程
```
Step 1: 订单确认
  - 双方确认订单详情
  - 签订销售合同
  - 发票

Step 2: 定金支付
  - 客户支付30%定金
  - 提供银行水单
  - 确认到账（2-3天）

Step 3: 开始生产
  - 收到定金后开始生产
  - 定期更新生产进度
  - 生产完成后拍照确认

Step 4: 余款支付
  - 客户支付70%余款
  - 提供银行水单
  - 确认到账（2-3天）

Step 5: 发货
  - 收到余款后发货
  - 提供物流单号
  - 跟踪货物运输
```

### PayPal付款流程
```
Step 1: 订单确认
  - 双方确认订单详情
  - 发送PayPal账单

Step 2: 客户付款
  - 客户通过PayPal付款
  - 即时到账通知

Step 3: 开始生产

Step 4: 发货
  - 直接发货（无需余款）
  - 提供物流单号
  - 跟踪货物运输
```

## 📜 贸易条款

### Incoterms 2020

#### EXW（工厂交货）
```
卖方责任：
- 生产产品
- 在工厂交货
- 提供商业发票

买方责任：
- 从工厂提货
- 支付运费
- 办理出口报关
- 承担所有费用和风险

适用场景：
- 客户自有物流
- 客户自有报关行
- 降低买方成本
```

#### FOB（离岸价）
```
卖方责任：
- 生产产品
- 装船至船上
- 办理出口报关
- 提供清关文件

买方责任：
- 支付运费
- 支付保险费
- 办理进口报关
- 承担装船后风险

适用场景：
- 海运大批量订单
- 买方安排船期
- 常用条款
```

#### CIF（成本、保险加运费）
```
卖方责任：
- 生产产品
- 支付运费至目的港
- 办理出口报关
- 购买保险（最低险别）
- 提供清关文件

买方责任：
- 办理进口报关
- 支付进口关税
- 承担卸货后风险

适用场景：
- 海运大批量订单
- 卖方安排物流
- 买方省心
```

#### DDP（完税后交货）
```
卖方责任：
- 生产产品
- 支付所有运费
- 办理进出口报关
- 支付所有关税
- 交付至买方指定地点

买方责任：
- 交付前验收
- 承担卸货责任

适用场景：
- 小批量空运
- 买方不熟悉进口流程
- 一站式服务
```

### 条款对比
| 条款 | 运费 | 保险 | 报关 | 关税 | 风险转移点 |
|------|------|------|------|------|------------|
| **EXW** | 买方 | 买方 | 买方 | 买方 | 工厂 |
| **FOB** | 买方 | 买方 | 卖方 | 买方 | 装船 |
| **CIF** | 卖方 | 卖方 | 卖方 | 买方 | 装船 |
| **DDP** | 卖方 | 卖方 | 卖方 | 卖方 | 交付地 |

### 进口清关
```
协助文件：
✓ 商业发票
✓ 装箱单
✓ 提单/运单
✓ 原产地证明
✓ 质量检测报告

客户需提供：
✓ 进口报关委托书
✓ 进口许可证（如需要）
✓ 特殊产品资质（如需要）
```

### 关税说明
```
关税税率：
- 美国：0-10%（根据HS编码）
- 欧盟：0-10%（根据HS编码）
- 中东：0-5%（根据HS编码）
- 亚太：0-15%（根据HS编码）

HS编码示例：
- 水晶烛台：7013.99.00
- 水晶装饰品：7018.99.00
- 水晶礼品：7013.37.00

关税计算：
- 关税 = 关税税率 × 海关价值
- 海关价值 = CIF价格
- 需咨询当地海关确认
```
