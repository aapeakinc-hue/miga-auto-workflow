## 📱 1. 产品照片收集（成本：0元）

### 拍摄工具
```
✓ 智能手机（iPhone/安卓）
✓ 自然光（靠窗位置）
✓ 简单背景（白色/黑色布或纸）
✓ 手机支架（可选，稳定拍摄）
```

### 拍摄步骤

#### 步骤1：准备拍摄环境
```
□ 选择光线充足的位置（靠窗，避免直射阳光）
□ 准备干净、简洁的背景（白色/黑色布）
□ 清洁产品（擦拭干净，去除指纹和灰尘）
□ 准备一块反光板（可选，白色纸板即可）
```

#### 步骤2：拍摄产品照片
```
□ 正面拍摄（展示产品全貌）
□ 侧面拍摄（展示产品轮廓）
□ 顶部拍摄（展示产品顶部设计）
□ 细节拍摄（展示雕刻、刻印、工艺）
□ 多角度拍摄（3-5个不同角度）

每个产品拍摄5-10张照片，选择最好的2-3张
```

#### 步骤3：拍摄应用场景
```
□ 烛台：点燃蜡烛，拍摄光影效果
□ 奖杯：放置在展示架上，拍摄整体效果
□ 装饰品：放置在桌面上，拍摄实际摆放效果
□ 礼品：拍摄包装后的效果
```

#### 步骤4：批量拍摄
```
□ 为每个产品拍摄照片
□ 为每个产品线拍摄组合照
□ 为所有产品拍摄集体照（可选）
□ 命名文件：产品名_角度_日期.jpg
```

### 照片要求
```
✓ 高清（至少2000px宽）
✓ 清晰（不模糊）
✓ 光线均匀（不过曝、不欠曝）
✓ 背景简洁（突出产品）
✓ 多角度（全方位展示）
```

### 免费照片编辑工具
```
在线工具：
- Canva (www.canva.com) - 免费，简单易用
- Fotor (www.fotor.com) - 免费，快速编辑
- Remove.bg (www.remove.bg) - 免费，去除背景

手机APP：
- Snapseed (免费) - 专业照片编辑
- Lightroom Mobile (免费版) - 轻度编辑
- VSCO (免费版) - 滤镜编辑

编辑要点：
- 调整亮度、对比度
- 裁剪到合适尺寸
- 去除背景（可选）
- 添加产品名称（可选）
```

## 🎥 2. 产品视频收集（成本：0元）

### 拍摄工具
```
✓ 智能手机（iPhone/安卓）
✓ 手机支架（稳定拍摄）
✓ 自然光（靠窗位置）
✓ 简单背景（白色/黑色布）
```

### 拍摄步骤

#### 视频类型1：产品展示视频（30-60秒）
```
□ 开场：产品全貌（5秒）
□ 多角度：旋转产品展示（10秒）
□ 细节：雕刻、刻印特写（10秒）
□ 应用：实际使用场景（10秒）
□ 结尾：产品特写（5秒）

拍摄要点：
- 稳定拍摄（使用支架）
- 缓慢旋转（每秒90度）
- 光线充足（避免阴影）
- 背景简洁（突出产品）
```

#### 视频类型2：工艺展示视频（60-90秒）
```
□ 原材料：展示水晶原材料（10秒）
□ 切割：展示切割过程（15秒）
□ 打磨：展示打磨过程（15秒）
□ 抛光：展示抛光过程（15秒）
□ 雕刻：展示雕刻过程（15秒）
□ 成品：展示成品效果（10秒）

拍摄要点：
- 获得工厂同意拍摄
- 尊重工厂隐私
- 可以说"我们的合作伙伴工厂"
```

#### 视频类型3：质检流程视频（30-60秒）
```
□ 开场：质检员工作场景（5秒）
□ 外观检查：检查产品外观（15秒）
□ 尺寸测量：测量产品尺寸（10秒）
□ 包装检查：检查包装质量（10秒）
□ 结尾：合格产品堆放（5秒）

拍摄要点：
- 展示专业性
- 强调100%全检
- 突出质量控制
```

### 免费视频编辑工具
```
在线工具：
- Canva (www.canva.com) - 免费，简单视频编辑
- Kapwing (www.kapwing.com) - 免费，在线编辑
- Clipchamp (www.clipchamp.com) - 免费，微软出品

手机APP：
- InShot (免费版) - 强大易用
- CapCut (免费版) - 字节跳动出品
- VN Video Editor (免费版) - 专业功能

编辑要点：
- 剪辑多余片段
- 添加背景音乐（可选）
- 添加产品名称（可选）
- 导出为MP4格式
- 控制文件大小（<100MB）
```

□ 列出所有合作过的客户

  - 所在地区
