## 🎯 信任度评分标准

### 评分标准
```
优秀（9-10分）：
- 完成所有项目
- 各方面都很完善
- 客户信任度极高

良好（7-8分）：
- 完成大部分项目
- 核心要素完善
- 客户信任度较高

中等（5-6分）：
- 完成部分项目
- 有基本要素
- 客户信任度一般

较差（3-4分）：
- 完成少量项目
- 缺少核心要素
- 客户信任度较低

差（0-2分）：
- 几乎未完成
- 缺少基本要素
- 客户信任度很低
```

### 评分计算
```
总分 = 100分

评分规则：
- 每个检查项完成后得相应分值
- 权重高的项目分值更高
- 完成度越高，得分越高

信任度评分 = 实际得分 / 100 × 10

例如：
- 实际得分：75分
- 信任度评分：7.5/10
```
