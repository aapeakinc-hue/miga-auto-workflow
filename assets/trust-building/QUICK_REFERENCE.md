## 📊 客户类型快速参考

### 批发商（Wholesaler）
```
特征：大批量、长期合作、价格敏感
年采购量：$100K-$300K
订单频率：每季度
关键需求：价格优势、质量稳定、低MOQ
最佳关键词：crystal gifts wholesale + [country]
```

### 酒店供应商（Hotel Supplier）
```
特征：高端市场、定制需求、单笔订单大
年采购量：$200K-$500K
订单频率：按项目
关键需求：定制logo、奢华包装、质量保证
最佳关键词：hotel supplies + [city]
```

### 活动策划（Event Planner）
```
特征：快速交付、定制设计、创新产品
年采购量：$20K-$100K
订单频率：按项目
关键需求：快速交付、3D激光雕刻
最佳关键词：event planning + [city]
```

### 礼品商（Gift Merchant）
```
特征：季节性、节日需求、精美包装
年采购量：$30K-$150K
订单频率：季节性（Q3-Q4）
关键需求：节日主题、精美包装、高质量
最佳关键词：crystal gifts + [country]
```

## 🎯 关键成功因素

### ✅ 成功要素

1. **精准关键词**：使用高转化关键词
2. **个性化邮件**：根据客户类型定制
3. **持续跟进**：发送3-7封跟进邮件
4. **快速响应**：24小时内回复客户
5. **数据驱动**：根据数据优化策略

### ❌ 常见错误

1. **关键词太宽泛**：避免使用"crystal"这样的宽泛词
2. **发送垃圾邮件**：避免群发，注重个性化
3. **缺乏跟进**：只发送一封邮件，缺乏跟进
4. **忽视客户类型**：不区分批发商、酒店、活动策划
5. **不跟踪效果**：不记录打开率、回复率
