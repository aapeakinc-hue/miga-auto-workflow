# 🚀 快速部署指南

## 立即部署（5分钟）

### 方式 1: Cloudflare Pages 直接上传（推荐）

1. **打开 Cloudflare**
   ```
   https://dash.cloudflare.com
   ```

2. **创建 Pages 项目**
   - 左侧菜单 → Workers & Pages
   - Create application → Pages
   - Create a project

3. **上传文件**
   - 选择 "Upload assets"
   - 拖入 `cloudflare-deploy/` 整个文件夹
   - 等待上传完成

4. **部署**
   - 点击 "Deploy site"
   - 等待 2-5 分钟
   - 获取网站 URL

5. **测试**
   - 访问生成的 URL
   - 测试所有功能
