# 📥 使用WinSCP下载并部署网站

## 🔧 WinSCP详细操作步骤

### 步骤1: 连接到服务器

1. **打开WinSCP**
   - 双击WinSCP图标启动程序

2. **新建站点**
   - 点击"新建站点"按钮（或按Ctrl+N）

3. **填写连接信息**
   ```
   文件协议: SFTP
   主机名: 您的服务器IP地址
   端口: 22
   用户名: root
   密码: 您的密码
   ```

4. **保存并连接**
   - 点击"保存"按钮
   - 给站点起个名字（如"MIGA服务器"）
   - 点击"登录"按钮

5. **首次连接提示**
   - 如果弹出"服务器主机密钥"提示，点击"是"
   - 连接成功后会看到左右两个窗口

---

### 步骤2: 下载文件

1. **找到文件**
   - 在右侧窗口（服务器端）
   - 导航到: `/workspace/projects/`
   - 找到文件: `miga-website-deploy.tar.gz`

2. **选择下载位置**
   - 在左侧窗口（本地端）
   - 导航到您想保存的文件夹（如桌面或下载文件夹）

3. **下载文件**
   - 右键点击 `miga-website-deploy.tar.gz`
   - 选择"下载"
   - 或直接将文件从右侧拖拽到左侧
   - 等待下载完成（16KB很快）

4. **验证下载**
   - 在左侧窗口看到文件已下载
   - 文件大小应该是16KB

---

### 步骤3: 解压文件

#### Windows系统（推荐使用7-Zip或WinRAR）

**方法1: 使用7-Zip**
1. 右键点击 `miga-website-deploy.tar.gz`
2. 选择"7-Zip" → "解压到当前文件夹"
3. 得到 `miga-website-deploy.tar` 文件
4. 再次右键点击 `miga-website-deploy.tar`
5. 选择"7-Zip" → "解压到当前文件夹"
6. 得到 `cloudflare-deploy` 文件夹

**方法2: 使用WinRAR**
1. 右键点击 `miga-website-deploy.tar.gz`
2. 选择"解压到当前文件夹"
3. 如果只解压出tar文件，再解压一次

**方法3: 使用Windows内置解压**
1. 右键点击文件 → "全部解压"
2. 如果不支持tar.gz格式，请安装7-Zip

#### 验证解压

解压后应该看到 `cloudflare-deploy` 文件夹，包含:
```
cloudflare-deploy/
├── index.html
├── products.html
├── images/
├── README.md
├── IMAGE_PREPARATION_CHECKLIST.md
└── UPLOAD_TO_CLOUDFLARE.md
```

---

### 步骤4: 上传到 Cloudflare Pages

#### 4.1 登录 Cloudflare

1. 打开浏览器
2. 访问: https://dash.cloudflare.com/
3. 使用您的Cloudflare账号登录

#### 4.2 进入 Workers & Pages

1. 在左侧菜单找到 **Workers & Pages**
2. 点击进入

#### 4.3 上传文件

**情况A: 您已经有 Pages 项目**

1. 在项目列表中找到 miga.cc 对应的项目
2. 点击项目名称进入
3. 点击 **Create deployment** 或 **Upload assets** 按钮
4. 拖拽 `cloudflare-deploy` 文件夹到上传区域
5. 等待上传完成
6. 点击 **Deploy**

**情况B: 您还没有 Pages 项目**

1. 点击右上角的 **Create application** 按钮
2. 页面显示两个选项
3. 点击 **Pages** 标签（默认已选中）
4. 点击 **Upload assets** 按钮
5. 拖拽 `cloudflare-deploy` 文件夹到上传区域
6. 填写项目名称: `miga-website`
7. 点击 **Deploy**

#### 4.4 等待部署

- 部署通常需要 **30秒 - 2分钟**
- 您会看到上传和部署进度
- 部署成功后会显示绿色 ✅

---

### 步骤5: 配置自定义域名

1. **进入域名设置**
   - 在项目页面，点击顶部的 **Custom domains** 标签

2. **添加域名**
   - 点击 **Set up a custom domain** 按钮
   - 输入您的域名: `miga.cc`
   - 点击 **Continue** 按钮

3. **确认DNS配置**
   - Cloudflare会自动显示DNS配置信息
   - 类型: CNAME
   - 名称: `@`（根域名）
   - 目标: `miga-website.pages.dev`（或其他项目名）
   - 确认信息正确

4. **激活域名**
   - 点击 **Activate domain** 按钮
   - 等待DNS生效（通常1-5分钟）

## 🔧 常见问题

### 问题1: WinSCP连接失败

**可能原因**:
- 服务器IP错误
- 用户名或密码错误
- SSH端口未开放

**解决方案**:
1. 检查服务器IP是否正确
2. 检查用户名（通常是root）
3. 检查密码是否正确
4. 确认SSH端口是22

### 问题2: 找不到文件

**可能原因**:
- 路径不正确
- 文件被删除或移动

**解决方案**:
1. 确认路径: `/workspace/projects/`
2. 在根目录搜索: Ctrl+F 搜索文件名

### 问题3: 解压失败

**可能原因**:
- 没有安装解压软件
- 文件损坏

**解决方案**:
1. 安装7-Zip: https://www.7-zip.org/
2. 重新下载文件

### 问题4: Cloudflare上传失败

**可能原因**:
- 文件夹结构错误
- 网络问题

**解决方案**:
1. 确认上传的是 `cloudflare-deploy` 文件夹（不是压缩包）
2. 检查网络连接
3. 重新上传

### 问题5: 网站无法访问

**可能原因**:
- DNS未生效
- 域名配置错误

**解决方案**:
1. 等待5-10分钟再试
2. 检查DNS配置
3. 使用隐身模式测试
