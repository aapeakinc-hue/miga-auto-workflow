## 🔄 如何更新目录册

### 方法1: 重新生成
```bash
# 1. 运行产品数据收集脚本
python create_product_catalog.py

# 2. 生成新的PDF
python generate_catalog_pdf.py

# 3. 下载新的PDF文件
cat assets/catalog_pdf_url.txt
```

### 方法2: 手动更新
1. 修改 `assets/catalog_data.json` 文件
2. 重新运行 `python generate_catalog_pdf.py`
3. 下载新的PDF文件
