## 📷 图片采购建议

### 方案A: 拍摄真实产品图片（推荐）

**优势**：
- ✅ 100%真实，展示实际产品质量
- ✅ 版权清晰，无法律风险
- ✅ 可以突出品牌特色
- ✅ 更容易获得客户信任

**拍摄建议**：
1. **设备**：
   - 使用单反相机或专业相机
   - 或使用高端手机（iPhone 12+/华为Mate 40+）

2. **背景**：
   - 使用白色或浅灰色背景布
   - 或使用渐变背景

3. **光线**：
   - 使用柔光箱或反光板
   - 自然光或专业摄影灯光
   - 避免直接闪光

4. **构图**：
   - 产品居中，留出适当留白
   - 多角度拍摄（正面、侧面、45度）
   - 拍摄细节特写

5. **后期处理**：
   - 调整亮度、对比度
   - 裁剪到800x600px
   - 保存为JPEG格式（质量80-90%）
   - 文件大小控制在100-300KB

**拍摄清单**：
- [ ] 每个产品至少3张照片（正面、侧面、细节）
- [ ] 确保光线充足且均匀
- [ ] 背景干净无杂物
- [ ] 产品清晰锐利
- [ ] 文件命名规范

## 📐 图片技术规格

### 尺寸要求

| 用途 | 宽度 | 高度 | 宽高比 |
|------|------|------|--------|
| 产品展示 | 800px | 600px | 4:3 |
| 产品详情 | 400px | 300px | 4:3 |
| 缩略图 | 200px | 150px | 4:3 |

### 文件格式

**推荐**：
- ✅ JPEG（首选）- 压缩率高，兼容性好
- ✅ PNG（透明背景需要时）- 无损压缩
- ✅ WebP（现代格式）- 文件更小

**不推荐**：
- ❌ BMP - 文件太大
- ❌ GIF - 只支持256色

### 文件大小

- **最大**: 500KB
- **推荐**: 100-300KB
- **最小**: 50KB

### 命名规范

**推荐格式**：
```
产品类型-编号.jpg
例如：
crystal-candle-holder-001.jpg
luxury-candelabra-002.jpg
crystal-tealight-003.jpg
```

**避免**：
```
❌ IMG_20240101_123456.jpg
❌ 产品图片1.jpg
❌ 123456.jpg
```

```bash

# 2. 将图片复制到目录
# 方法A: 使用文件管理器拖拽
# 方法B: 使用命令行
cp /path/to/your/images/*.jpg cloudflare-deploy/images/

ls -lh cloudflare-deploy/images/
```
