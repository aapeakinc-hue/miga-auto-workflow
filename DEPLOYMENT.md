## Deployment Options

### Option A: Manual Upload (Recommended - Fastest)

**Time**: 5 minutes

1. Go to Cloudflare Pages Dashboard: https://dash.cloudflare.com
2. Navigate to: Pages > Create a project
3. Select "Upload assets"
4. Drag and drop the entire `cloudflare-deploy/` folder
5. Click "Deploy Site"
6. Wait 2-5 minutes for deployment
7. Get your website URL (e.g., `https://your-site.pages.dev`)
8. Add custom domain `https://miga.cc` (optional)

### Option B: GitHub Integration (Recommended for Updates)

**Time**: 15 minutes setup, instant updates thereafter

1. Initialize Git repository:
   ```bash
   cd cloudflare-deploy
   git init
   git add .
   git commit -m "English website launch"
   ```

2. Push to GitHub (create repository first)

3. Connect Cloudflare Pages:
   - Go to Cloudflare Pages Dashboard
   - Select "Connect to Git"
   - Choose your repository
   - Configure:
     - Build command: (leave empty)
     - Build output directory: `/`
   - Click "Save and Deploy"

4. Every time you push changes, Cloudflare will automatically redeploy
